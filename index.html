Prompt

The instruction you give to AI

▼
Think of it like...

A prompt is like placing an order at a restaurant. The more specific your order ("grilled salmon, medium, no sauce"), the better the result. Vague orders ("just bring me food") get unpredictable results.

// A prompt is just text you send to AI
prompt = "Summarize this article in 3 bullet points"
Concept unlocked!
🔌

API

How your code talks to AI

▼
Think of it like...

An API is like a waiter at a restaurant. You (your code) don't walk into the kitchen (Claude's brain). Instead, you hand your order to the waiter (the API), and they bring back the result.

// You send a message, the API sends a reply
response = ask_claude("What's a fun weekend plan?")
Concept unlocked!
⚙️

Function

A reusable set of instructions

▼
Think of it like...

A function is like a recipe card. You write the steps once ("make pancakes"), and then anytime you want pancakes, you just say "follow the pancake recipe" instead of reciting every step again.

// Write once, use anywhere
function summarize(text) {
  return ask_claude(`Summarize: ${text}`)
}
Concept unlocked!
